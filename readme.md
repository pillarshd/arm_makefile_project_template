# makefile template project

这是针对arm cortex-M 系列MCU的模板工程。

## 使用的工具

- [GNU Arm Embedded Toolchain](https://developer.arm.com/downloads/-/gnu-rm)
- [pyocd](https://github.com/pyocd/pyOCD)
- make
- vscode，推荐插件：
    - Cortex-Debug
    - C/C++
    - Doxygen Documentation Generato

## 目前支持

- APM:
    - APM32F103
- air(由于涨价，暂时不更新):
    - air001
    - air32F103XB
